    Agregar nueva salida a Terreno


    Luego debera rellenar el formulario de salida con todos los datos que se solicitan, cuando ya tenga lo datos ingresados, revise cautelosamente para listar los datos deseados. Para finalizar presione el boton:

  • Modificar Salida a Terreno

    Para modificar algun dato de una Salida a Terreno debe dirigirse Gestionar Salidas a terreno y seleccionar la salida que desea modificar y presionar el boton: Editar


    Luego de presionarle boton, la pagina lo redireccionara a un lista con los datos ingresados, usted debera editar los datos los cuales desea modidifcar, finalrmente debera presionar ael boton:

  • Eliminar Salida a Terreno

    Para eliminar una Salida a terreno dirigirse Gestionar Salidas a terreno y seleccionar la salida que desea eliminar y precionar el boton: Eliminar
