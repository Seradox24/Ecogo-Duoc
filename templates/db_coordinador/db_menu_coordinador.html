Gestionar Salidas a Terreno

  • Crear nueva salida a terreno

    Para comenzar con las gestion de las salidad a terreno, debera presionar en el boton con el nombre de Agregar nueva Salida a Terreno:

    Agregar nueva salida a Terreno


    Luego debera rellenar el formulario de salida con todos los datos que se solicitan, cuando ya tenga lo datos ingresados, revise cautelosamente para listar los datos deseados. Para finalizar presione el boton:

  • Modificar Salida a Terreno

    Para modificar algun dato de una Salida a Terreno debe dirigirse Gestionar Salidas a terreno y seleccionar la salida que desea modificar y presionar el boton: Editar


    Luego de presionarle boton, la pagina lo redireccionara a un lista con los datos ingresados, usted debera editar los datos los cuales desea modidifcar, finalrmente debera presionar ael boton:

  • Eliminar Salida a Terreno

    Para eliminar una Salida a terreno dirigirse Gestionar Salidas a terreno y seleccionar la salida que desea eliminar y precionar el boton: Eliminar


    Luego de ello aparecera un mensaje emergente preguntando si esta segore de realizar la eliminarcon de la Salida a Terreno, ahi debera elegir una opcion, ya sea: No, Cancelar o Sí, Eliminar

  • Semaforo de Salida

    Para selecciona un estado para el semaforo de la Salida a Terreno debe dirigirse Gestionar Salidas a terreno y seleccionar El semaforo que desea definir o modificar seleccionando el icono:

         

    Al momento de presionar un de los iconos, la pagina redireccionara a un sitio donde se tiene que definir el estado del Semaforo, luego de ello presione el boton:


Gestionar Asignaturas

  • Agregar Asignatura

    Para agregar una nueva asignatura debe dirigirse a gestionar asignaturas y presionar el boton: Agregar Asignatura


    Luego de ello la pagina lo redireccionara a un formulario, el cual debe rellenar con Nombre, Sigla, Docentes y Secciones. Despues de ingresar los datos debera presionar el boton:

  • Editar Asignatura

    Para modificar algun dato de una Asignatura debe dirigirse a gestionar asignaturas y seleccionar la asignatura que desea modificar y presionar el boton: Editar


    Luego de presionarle boton, la pagina lo redireccionara a un lista con los datos ingresados, usted debera editar los datos los cuales desea modidifcar, finalmente debera presionar el boton:

  • Eliminar Asignatura

    Para eliminar una Asignatura dirigirse a gestionar asignaturas y seleccionar la asignatura que desea eliminar y precionar el boton: Eliminar


    Luego de ello aparecera un mensaje emergente preguntando si esta seguro de realizar la eliminarcon de la Asignatura, ahi debera elegir una opcion, ya sea: No, Cancelar o Sí, Eliminar


Gestion de Usuarios

  • Sistema de busqueda

    Cuando este en el mueno de Gestion de Usuarios debe dirigirse a la barra superior de la pagina y seleccionar entre las opciones para filtrar un usuario o usuarios en especifico:


    Tambien puede filtrar mediante palabras claves en la barra de busqueda, para un usuarios con un dato especifico.
