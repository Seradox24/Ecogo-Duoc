Manual de Usuario Coordinador


Gestionar Salidas a Terreno

  • Crear nueva salida a terreno

    Para comenzar con la gestión de las salidas a terreno, deberá presionar el botón con el nombre de "Agregar nueva Salida a Terreno":

    Agregar nueva salida a Terreno


    Luego deberá rellenar el formulario de salida con todos los datos solicitados. Cuando ya tenga los datos ingresados, revíselos cautelosamente para listar los datos deseados.

  • Modificar Salida a Terreno

    Para modificar algún dato de una Salida a Terreno, debe dirigirse a Gestionar Salidas a terreno y seleccionar la salida que desea modificar. Luego, presione el botón:

    Editar


    Después de presionar el botón, la página lo redireccionará a una lista con los datos ingresados. Usted deberá editar los datos que desea modificar. Finalmente, presione el botón:

  • Eliminar Salida a Terreno

    Para eliminar una Salida a Terreno, diríjase a Gestionar Salidas a terreno y seleccione la salida que desea eliminar. Luego, presione el botón:

    Eliminar


    Después de ello, aparecerá un mensaje emergente preguntando si está seguro de realizar la eliminación de la Salida a Terreno. Deberá elegir una opción, ya sea: No, Cancelar o Sí, Eliminar.

  • Semaforo de Salida

    Para seleccionar un estado para el semáforo de la Salida a Terreno, diríjase a Gestionar Salidas a terreno y seleccione el semáforo que desea definir o modificar seleccionando el icono:

         


    Al momento de presionar uno de los iconos, la página redireccionará a un sitio donde se tiene que definir el estado del semáforo. Luego de ello, presione el botón:


Gestionar Asignaturas

  • Agregar Asignatura

    Para agregar una nueva asignatura, debe dirigirse a gestionar asignaturas y presionar el botón:

    Agregar Asignatura


    Luego de ello, la página lo redireccionará a un formulario, el cual debe rellenar con Nombre, Sigla, Docentes y Secciones. Después de ingresar los datos, deberá presionar el botón:

  • Editar Asignatura

    Para modificar algún dato de una asignatura, debe dirigirse a gestionar asignaturas y seleccionar la asignatura que desea modificar. Luego, presione el botón:

    Editar


    Después de presionar el botón, la página lo redireccionará a una lista con los datos ingresados. Usted deberá editar los datos que desea modificar. Finalmente, deberá presionar el botón:

  • Eliminar Asignatura

    Para eliminar una asignatura, diríjase a gestionar asignaturas y seleccione la asignatura que desea eliminar. Luego, presione el botón:

    Eliminar


    Luego de ello, aparecerá un mensaje emergente preguntando si está seguro de realizar la eliminación de la asignatura. Ahí deberá elegir una opción, ya sea: No, Cancelar o Sí, Eliminar.


Gestión de Usuarios

  • Sistema de búsqueda

    Cuando esté en el menú de Gestión de Usuarios, diríjase a la barra superior de la página y seleccione entre las opciones para filtrar un usuario o usuarios en específico:


    También puede filtrar mediante palabras clave en la barra de búsqueda, para un usuario con un dato específico.

  • Agregar Usuario

    Para agregar un nuevo usuario, debe dirigirse a Gestión de Usuarios y presionar el botón:

    Agregar Usuario

    Luego deberá completar el formulario con todos los datos solicitados. Después de ingresar los datos, revíselos cuidadosamente para lograr una creación de usuario exitosa. Para finalizar, presione el botón:

  • Editar Usuario

    Para modificar algún dato del usuario, debe dirigirse a Gestión de Usuarios y seleccionar el usuario que desea modificar. Luego, presione el botón:

    Editar


    Después de presionar el botón, la página lo redireccionará a una lista con los datos ingresados. Usted deberá editar los datos que desea modificar. Finalmente, presione el botón:

  • Eliminar Usuario

    Para eliminar un usuario, debe dirigirse a Gestión de Usuarios y seleccionar el usuario que desea eliminar. Luego, presione el botón:

    Eliminar


    Luego de ello, aparecerá un mensaje emergente preguntando si está seguro de realizar la eliminación del usuario. Ahí deberá elegir una opción, ya sea: No, Cancelar o Sí, Eliminar.

    Para visualizar la información subida por el usuario, en este caso, la ficha médica, debe dirigirse a Gestión de Usuarios, seleccionar el usuario que desea y presionar el botón:


    Al presionar el botón se desplegará un enlace que contiene la ficha médica subida por el usuario.
